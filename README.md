# DYQ-Blog

这是一个基于GitHub Pages搭建的个人展示博客网站，用于展示个人信息、项目经历和竞赛成果。

## 网站内容

- **个人信息展示** - 基本信息、联系方式等
- **竞赛科创栏目**
  - 3D打印项目
  - 单片机课程设计
  - 工创赛
  - 机创赛
  - 智能装备设计大赛
  - 朱雀3D打印机组装
  - 科普活动
- **个人简历** - PDF预览和下载功能

## 功能特性

### 图片预览功能

- 点击图片可打开全屏预览
- 鼠标滚轮缩放（以鼠标位置为原点）
- 快捷键旋转（`[` 逆时针90°，`]` 顺时针90°）
- ESC键退出预览

### 视频播放控制

- 点击播放按钮开始播放
- 同一时间只能有一个视频播放
- 切换页面时自动停止所有视频并复位

### 个人简历

- 在线预览PDF简历
- 一键下载简历

## 本地预览

1. 克隆或下载本仓库到本地
2. 使用浏览器直接打开 `index.html` 文件即可预览

## 目录结构

```
DYQ-Blog/
├── index.html          # 主页面文件
├── README.md          # 说明文档
└── media/            # 媒体资源文件夹
    ├── *.jpg         # 项目图片
    ├── *.mp4         # 项目视频
    └── 简历.pdf      # 个人简历PDF
```

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 响应式设计

## 快捷键

| 快捷键  | 功能               |
| ------- | ------------------ |
| ESC     | 退出图片预览       |
| [ 或 ← | 图片逆时针旋转90° |
| ] 或 → | 图片顺时针旋转90° |
