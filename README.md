# 个人展示博客

这是一个基于GitHub Pages搭建的个人展示博客网站，用于展示个人信息和项目经历。

## 网站内容

- 个人信息展示
- 3D打印项目
- 单片机课程设计
- 各类竞赛项目（工创赛、机创赛等）
- 智能装备设计大赛
- 朱雀3D打印机组装

## 本地预览

1. 克隆或下载本仓库到本地
2. 使用浏览器直接打开 `index.html` 文件即可预览

## 部署到GitHub Pages

### 方法一：使用GitHub仓库

1. 在GitHub上创建一个新仓库，命名为 `your-username.github.io`（将your-username替换为你的GitHub用户名）
2. 将本仓库的所有文件推送到新创建的仓库
3. 等待几分钟，访问 `https://your-username.github.io` 即可查看网站

### 方法二：使用现有仓库的gh-pages分支

1. 在现有仓库中创建 `gh-pages` 分支
2. 将所有文件推送到 `gh-pages` 分支
3. 访问 `https://your-username.github.io/repository-name` 即可查看网站

## 目录结构

```
DYQ-Blog/
├── index.html          # 主页面文件
├── README.md          # 说明文档
└── media/            # 媒体资源文件夹
    ├── *.jpg         # 项目图片
    └── *.mp4         # 项目视频
```

## 维护与更新

如需更新个人信息或添加新项目，请直接编辑 `index.html` 文件，并将相关的图片或视频文件放入 `media` 文件夹中。
